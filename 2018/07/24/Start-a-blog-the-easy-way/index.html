Introduction

A website or a blog of your own is very cool right? Today I will introduce you the easist way to have your own webpage (as far as I know).

Here I chose Hexo(https://hexo.io) and github(https://github.org), which will allow you to create a blog and have the freedom to customize the themes and styles of it.

Start a github page

Creating a github account

You need a Github account to start with, which is quite simple with just a visitto their website.

Creating a repo

You need a new repository for your blog. To create a new repo, clik the new button.

Then you need to set your repo initially. A veeeeeery important thing: The repo name is the same as your github name

After entering your repo name, go to your repo page and then click the setting button at the right side.

find the option GitHub Pages. and click Choose a theme

OK, till to now, you have finished the first step. Next we are going to prepare our tools for setting blog.

Download local tools

Since I use a Mac OS X operating system, so I am going to talk about the steps under Mac OS. If you use a Linux System, it is quite same. But if you use Windows OS, there may be a little different. But I believe it will not stop you.

  • Git
  • Node.js
