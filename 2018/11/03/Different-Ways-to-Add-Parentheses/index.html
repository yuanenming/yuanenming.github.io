Different Ways to Add Parentheses

2018-11-03
  • Algorithm
  • LeetCode
  1. Description
    1. Example
  2. Idea
  3. Code

Description

Given a string of numbers and operators, return all possible results from computing all the different possible ways to group numbers and operators. The valid operators are +, - and *.

Example

Idea

这个问题让我想到了之前看过的一个匹配n个括号的问题，此前的问题是利用括号的规则递归实现的，详见(https://www.yuanenming.cn/2018/09/22/Algorithm-Generate-Parentheses/) ，我觉得这不是DP也不是分治，类似二叉树遍历。参考此前一题，我按照先generate括号再填充数字运算符，最后计算的想法，想了好久，算法想清楚了，但是并不好实现。

跳出此前一题的思维，这个问题可以采用分治的思想（divide and conquer）。
对于一个split后的字符串，遍历其中的运算符，left op right，递归下去便是所有的优先级运算组合。
